● ORDER MANAGEMENT

Orders:           In order list admin will get all the information of his orders

All orders:          Admin can see all his orders by Selecting the Date Range .

Pending:          In the pending option admin can see all his pending orders by select

“Date Range”. If you want to accept your order from the pending list then click on the action button of the product after that you can accept your order from “Order & Shipping Info”. Now click on the drop down menu to select the “Confirmed” option, now the order has been accepted and go to the next stage.

Confirmed:               In the confirmed option, the admin can see all his confirmed

orders. Once the order has been confirmed, the order will be sent for the packaging. You can change the status of the product by clicking on the action button of that order. Now from “Order & Shipping Info” click on the drop down menu to select the “Packaging” option, now order will go for the next stage.

Packaging:       In this option you can see all the orders which are ready for packaging. Once the order has been packed it will go for shipping. Now click on the action button of the order then from “Order & Shipping Info” click on the drop down menu to select the “Out for Delivery” option, now the order will go to the next stage.

Out of delivery:        In this option you can see all the orders which are ready for delivery. Here you can assign a delivery man from “Shipping Type or Shipping Method” and you can choose the delivery type from the drop down menu “by self delivery man” or “third party service” also you can update the payment status of the prepaid order.

Delivered: In this option admin can see the product which is delivered. Once the order has been delivered. The delivery man will update the status of the product. Then you can change the status of the order from “Order & Shipping Info” click on the drop down menu to select the “Delivered” option and update the payment select to paid.

Returned:                  Here the admin can see all the returned orders. Click on the

returned button after you show the return product then you can change the status of the return product by an action button.

Failed to Deliver:     In the option admin can see which product has failed to deliver. If the delivery boy could not deliver the product at the time. Then You will try the order for delivery again the next day.

Canceled:            Here the admin can see all his cancel orders. If you want to see a

canceled order of a seller or inhouse product, then you can select the seller and date from the "select date range" option.

Refund request:

Pending:         In the option admin can see the customer’s refund request. If the refund request follows ”Refund Policy” you can approve the request. For approval click on the action button of the cancel order then click on ”Change refund status” now you can select the “Approved” option. Now this refund order has been approved.

Approved:      In the option admin can see the approved refund request. If you want to refund the order, then you will click on the action button. After this, select the refund option from "Change Refund Status" and select the payment method. If you want to reject the refund request you can select the “Rejected” option and give a reason for rejection of the request.

Refunded:        In the option admin can see all the refunded orders.

Rejected:          In the option admin can see all the rejected refund orders.

button after that you will fill the product name and the details of the product. After that scroll down you will see general information. Now you will fill product type, category, sub category, sub sub category, product code, brand, unit. Now you can fill the color of the product from the variation option.After that you will fill product price and stocks.You will fill these option are unit price, purchase price, tax percent, discount, discount type, total quantity, minimum order quantity, shipping cost, shipping cost multiply with quantity.After that you will fill product image and thumbnail image then submit.
